    Interface IChromiumBrowser

    Chromium-specific features including Tracing, service worker support, etc. You can use chromiumBrowser.StartTracingAsync and chromiumBrowser.StopTracingAsync to create a trace file which can be opened in Chrome DevTools or timeline viewer.
