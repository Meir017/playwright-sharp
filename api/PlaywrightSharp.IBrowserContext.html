Interface IBrowserContext

BrowserContexts provide a way to operate multiple independent browser sessions. If a IPage opens another page, e.g.with a window.open call, the popup will belong to the parent page's browser context. PlaywrightSharp allows creation of "incognito" browser contexts with NewContextAsync(BrowserContextOptions) method. "Incognito" browser contexts don't write any browsing data to disk.

Namespace: PlaywrightSharp
Assembly: PlaywrightSharp.dll
Syntax
public interface IBrowserContext : IAsyncDisposable
Examples
// Create a new incognito browser context
const context = await browser.NewContextAsync();
// Create a new page inside context.
const page = await context.NewPageAsync("https://example.com");
// Dispose context once it's no longer needed.
await context.CloseAsync();
